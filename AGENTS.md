# Expenses App Agent Notes

Always use bun

## Commands

```bash
bun install
bun run dev          # web + convex together
bun run dev:web      # Vite app only
bun run dev:convex   # Convex functions only
bun run test
bun run lint
bun run typecheck
bun run check        # lint + typecheck + format check
bun run build
bun run start
```
